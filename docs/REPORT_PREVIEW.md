- **Font Family**: Segoe UI, Tahoma, Geneva, Verdana, sans-serif

## 🎯 Interactive Features

### Hover Effects
- **Stat Cards**: Lift up 5px with enhanced shadow
- **Test Items**: Slide right 5px with background color change
- **Links**: Underline on hover

### Responsive Breakpoints
- **Desktop** (>768px): 4-column grid for stats
- **Tablet** (768px): 2-column grid
- **Mobile** (<768px): Single column, smaller fonts

## 📝 Test Item Cards

Each test displays:

```
┌─────────────────────────────────────────────────────┐
│ ✅ test_iftar_countdown_one_hour_remaining  [PASSED]│
│    tests/features/test_iftar_countdown.py           │
│                                          ⏱ 0.028s  │
└─────────────────────────────────────────────────────┘
```

**Elements:**
- Left border: 4px colored by status
- Icon: Status-specific (✅/❌/⚠️)
- Name: Bold, 1.1rem
- Path: Monospace font, gray
- Badge: Colored pill with status
- Duration: Clock icon + time

**Status Colors:**
- Passed: Green left border
- Failed: Red left border
- Skipped: Yellow left border
